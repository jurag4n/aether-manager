Fix R8 Kotlin metadata warning

Changed:
- gradle/libs.versions.toml: agp 8.6.1 -> 8.13.2

Reason:
- AGP 8.6.1 bundles R8 8.6.x, which supports Kotlin metadata up to Kotlin 2.1.
- Some dependencies in this project are compiled with newer Kotlin metadata, so R8 warns during minifyReleaseWithR8.
- AGP 8.13.2 bundles R8 8.13.19, which supports Kotlin 2.3 metadata and supports compileSdk 36.

Build command:
./gradlew clean :app:assembleRelease --no-configuration-cache --warning-mode all
