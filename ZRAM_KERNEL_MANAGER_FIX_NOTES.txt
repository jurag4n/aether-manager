Aether Manager - ZRAM + Kernel Manager Improvements

Fix utama:
- ZRAM 1GB sekarang benar-benar resize jika zram aktif tapi ukuran berbeda.
- Sebelumnya zram aktif langsung dianggap sukses, jadi memilih 1GB tidak mengubah ukuran bawaan ROM.
- Apply ZRAM sekarang: swapoff -> reset -> set algorithm -> set disksize -> mkswap -> swapon -> verify.
- Ada guard MemAvailable agar tidak resize saat RAM terlalu rendah.
- Jika resize gagal, engine mencoba restore ukuran ZRAM lama.
- Jika fitur ZRAM tidak aktif, engine tidak mematikan/reset zram bawaan ROM agar aman.
